# Contributing to OLLM Docs

Thanks for helping improve the documentation.

## Quickstart

1. Fork this repository
2. Create a branch
3. Edit or add `.mdx` files in `content/docs/ollm/`
4. Open a pull request

## Style Guidelines

- Use frontmatter with `title` and `description`
- Prefer Markdown for structure
- Use MDX components sparingly for callouts and examples

## Example Frontmatter

```mdx
---
title: "My Page"
description: "Short summary"
---
```
